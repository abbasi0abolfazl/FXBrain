## سیستم FX Brain

### جمع‌آوری داده‌های اقتصادی

#### **منابع داده:**
- ‏**ForexFactory:** داده‌های تقویم اقتصادی به صورت XML
- ‏**API‌های خبری:** Bloomberg, Reuters, CNBC
- **وب‌سایت‌های بانک‌های مرکزی:** FED, ECB, BOJ, BOE
- **رسانه‌های اقتصادی:** Financial Times, Wall Street Journal

#### **فرآیند جمع‌آوری:**

```python
from forexfactory_scraper import fetch_and_parse_xml

# دریافت رویدادهای اقتصادی با فیلتر دلار و تأثیر بالا
events = fetch_and_parse_xml(
    currency_filter="USD",
    impact_filter="High",
    user_timezone="Asia/Tehran"
)
```

**نمونه داده‌های جمع‌آوری شده:**
```json
[
    {
        "title": "FOMC Meeting Minutes",
        "currency": "USD",
        "date": "10-08-2025",
        "time": "21:26",
        "impact": "High",
        "forecast": null,
        "previous": null,
        "url": "https://www.forexfactory.com/calendar/304-us-fomc-meeting-minutes",
        "sessions": ["New York"]
    }
]
```

### استخراج اطلاعات تکمیلی

#### **وب‌اسکرپینگ با Selenium:**
```python
from selenium import webdriver
from bs4 import BeautifulSoup

def get_event_details(url):
    driver = webdriver.Firefox()
    driver.get(url)
    soup = BeautifulSoup(driver.page_source, 'html.parser')
    
    # استخراج توضیحات رویداد
    description = soup.find('p', class_="calendar-event__description").get_text()
    return description
```

**اطلاعات تکمیلی استخراج شده:**
```json
{
    "text": "It's a detailed record of the FOMC's most recent meeting, providing in-depth insights into the economic and financial conditions that influenced their vote on where to set interest rates;"
}
```

### API endpoints جدید برای AI

#### **تحلیل هوشمند:**
```
GET    /api/ai/analyze/event/{event_id}     - تحلیل یک رویداد خاص
POST   /api/ai/analyze/news                  - تحلیل خبر جدید
GET    /api/ai/predictions/{pair}            - پیش‌بینی برای جفت ارز
GET    /api/ai/sentiment/{currency}          - احساسات بازار برای یک ارز
POST   /api/ai/batch-analyze                  - تحلیل دسته‌ای رویدادها
```

#### **مدیریت داده‌ها:**
```
POST   /api/data/scrape/calendar             - اسکرپ تقویم اقتصادی
POST   /api/data/scrape/event/{url}          - اسکرپ جزئیات رویداد
GET    /api/data/events/{currency}           - دریافت رویدادهای یک ارز
GET    /api/data/processed/{date}            - داده‌های پردازش‌شده
```

### صفحات 

#### صفحه ورود (Login)
   - فرم ورود با ایمیل و رمز عبور
   - گزینه "مرا به خاطر بسپار"
   - حساب‌های آزمایشی (دمو، حرفه‌ای، مدیر)
   - لینک ثبت‌نام و فراموشی رمز

#### داشبورد اصلی (Dashboard)
   - نمایش خلاصه وضعیت بازار
   - ویجت‌های قابل شخصی‌سازی
   - آمار لحظه‌ای (سیگنال‌ها، رویدادها، هشدارها)
   - تیکر زنده قیمت‌ها
   - **ویجت جدید: تحلیل‌های AI روزانه**

#### صفحه اخبار (News)
   - اخبار لحظه‌ای بازارهای مالی
   - فیلتر بر اساس تأثیر (بالا/متوسط/پایین)
   - نمایش پیش‌بینی‌های هوشمند **با تحلیل AI**
   - تاریخچه اخبار با **امتیاز احساسات**

#### تقویم اقتصادی (Calendar)
   - رویدادهای مهم اقتصادی
   - فیلتر بر اساس کشور و اهمیت
   - نمایش زمان رویدادها
   - پیش‌بینی و مقادیر قبلی
   - **تحلیل AI برای هر رویداد**

#### سیگنال‌های معاملاتی (Signals)
   - سیگنال‌های خرید و فروش لحظه‌ای **بر اساس AI**
   - نمایش نقاط ورود، حد سود و حد ضرر
   - درصد اعتماد سیگنال 
   - تحلیل و دلایل سیگنال (تولید شده توسط AI)

#### هشدارها (Alerts)
   - ایجاد هشدارهای شخصی‌سازی شده
   - انواع هشدار (قیمت، RSI، اخبار مهم)
   - کانال‌های اعلان (وب، تلگرام، ایمیل)
   - مدیریت هشدارهای فعال/غیرفعال
   - **هشدارهای هوشمند AI**

#### حساب کاربری (Account)
   - مدیریت اطلاعات شخصی
   - تنظیمات امنیتی
   - مدیریت کلیدهای API
   - تنظیمات اعلان‌ها
   - **تنظیمات شخصی‌سازی AI**

### نتیجه‌گیری

سیستم هوشمند **FX Brain** با ترکیب **داده‌های لحظه‌ای اقتصادی** و **تحلیل پیشرفته AI**، یک پلتفرم قدرتمند برای معامله‌گران فراهم می‌کند. این سیستم قادر است:

- هزاران رویداد اقتصادی را در لحظه پایش کند
- اخبار را به زبان‌های مختلف تحلیل کند
- پیش‌بینی‌های دقیق از جهت‌گیری بازار ارائه دهد
- سیگنال‌های معاملاتی با دقت بالا تولید کند
- به معامله‌گران در تصمیم‌گیری‌های سریع و دقیق کمک کند

با استفاده از **مدل‌های زبانی پیشرفته** و **تکنیک‌های یادگیری ماشین**، FX Brain یک همراه هوشمند برای تمام معامله‌گران حرفه‌ای خواهد بود
